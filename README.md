# Calculadora 2026

![]()

## Introdução

Uma calculadora de console simples mas poderosa que permita realizar as quatro operações matemáticas, além de visualização do histórico de operações e tabuada.

## Funcionalidades

- **Operações básicas**: Realize somas, subtrações, multiplicações e divisões e divisões com facilidade.
- **Tratamento de Divisão Zero**: A calculadora é capaz de validar erros de divisão por zero.
- **Tabuada**: A calculadora é capaz de gerar tabuada de um numero informado.
- **Histórico de Operações**: A calculadora é capaz de armazenar um histórico das operações anteriores.

## Como utilizar o programa

1. Clone o repostiório ou baixe o código comprimido em .zip.
2. Abra o emulador de terminal e navegue até a pasta raiz.
3. Utilize o codigo abaixo para restaurar as dependências do projeto.

    ```
    dotnet restore
    ```

4. Em seguida compile e execute o projeto com comando:

    ```
    dotnet run --project Calculadora.ConsoleApp
    ```
